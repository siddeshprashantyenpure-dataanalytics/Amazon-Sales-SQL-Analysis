# 📊 Amazon Sales SQL Analysis

## 🚀 Project Overview

This project is an end-to-end SQL analysis of an Amazon Sales dataset containing over **128,000+ sales records**. The objective is to demonstrate practical SQL skills by performing data cleaning, data quality checks, exploratory analysis, business reporting, and advanced SQL analytics.

Rather than solving isolated SQL questions, this project follows a real-world analytical workflow similar to what a Data Analyst would perform in an organization.

---

# 🎯 Project Objectives

* Import and prepare raw sales data
* Perform data cleaning and quality validation
* Analyze sales performance across different business dimensions
* Identify revenue trends and customer purchasing patterns
* Evaluate courier performance and order status distribution
* Generate business insights using SQL
* Demonstrate intermediate to advanced SQL concepts

---

# 📂 Dataset Information

* **Dataset:** Amazon Sales Report
* **Records:** 128,000+ Orders
* **Columns:** 24
